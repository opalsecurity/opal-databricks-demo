# Opal-Databricks Notebook Demo

This repository hosts the files for [this blog post](https://opal.dev/blog/opal-with-databricks-notebooks).

Using your own Databricks Free Notebooks environment and your Opal instance's API key, the notebooks are designed to ingest data via your personal access token for data analysis and insights.
